# LightTrack - Remaining Work

Lightweight personal time tracker with export. Keep it simple.

## Core Functionality

- [x] **Tracking toggle** - Start/stop wired to ActivityTracker
- [x] **Activity display** - Shows tracked activities in feed
- [x] **Export** - CSV export with save dialog
- [x] **Data persistence** - Activities save via electron-store
- [x] **Manual time entry** - Add entries without tracking
- [x] **Edit/delete entries** - Fix mistakes
- [x] **Date filtering** - Filter activity list by date

## Out of Scope (Don't Build)

- ML/AI anything
- Third-party integrations (Jira, Slack, Calendar)
- Team features
- TypeScript migration
- GraphQL/REST APIs
- Platform-specific features (Touch Bar, Windows Timeline)
- Delta updates / staged rollouts
- 90% test coverage goals
- IndexedDB migration
- Virtual scrolling
